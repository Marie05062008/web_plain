Spiele Memory

Beim Memory-Spiel musst du Kartenpaare finden. Klicke auf zwei Karten, um sie umzudrehen. Wenn sie übereinstimmen, bleiben sie sichtbar. Finde alle Paare, um zu gewinnen!
